    JavaScript 中，this 在不同应用场景，如何取值？
    2019-11-20 · 吾儿滨滨

    一、作为普通函数

    function printThis() {
    console.log(this)
}
printThis()

    Window {parent: Window, postMessage: ƒ, blur: ƒ, focus: ƒ, close: ƒ, …}

    二、使用 call apply bind

    function fn1() {
    console.log(this)
}
fn1.call({ x: 100 })  // 100
const fn2 = fn1.bind({ x: 200 })
fn2()  // 200

    作用：改变函数执行时的上下文，即改变函数运行时的this指向。

    • call apply 和 bind 的区别：

    call 和 apply 改变了函数的 this 上下文后便执行该函数,而 bind 则是返回改变了上下文后的一个函数。

    • call 和 apply 的区别：

    他们俩之间的差别在于参数的区别，call 和 apply 的第一个参数都是要改变上下文的对象，而 call 从第二个参数开始以参数列表的形式展现，apply 则是把除了改变上下文对象的参数放在一个数组里面作为它的第二个参数。

    let arr = [2, 5, 9]
console.log(Math.max.call(null, 2, 5, 9))  // 9
console.log(Math.max.call(null, arr))  // NaN
console.log(Math.max.apply(null, arr))  // 9

    三、作为对象方法被调用

    const people = {
    name: "Merry",
    sayHi() {
        console.log(this)
    },
    wait() {
        setTimeout(function () {
            console.log(this)
        })
    }
}
people.sayHi()
people.wait()

    {name: “Merry”, sayHi: ƒ, wait: ƒ}

    Window {parent: Window, postMessage: ƒ, blur: ƒ, focus: ƒ, close: ƒ, …}

    四、箭头函数

    const people = {
    name: "Merry",
    sayHi() {
        console.log(this)
    },
    wait() {
        setTimeout(() => {
            console.log(this)
        })
    }
}
people.sayHi()
people.wait()

    {name: “Merry”, sayHi: ƒ, wait: ƒ}

    {name: “Merry”, sayHi: ƒ, wait: ƒ}

    五、在 class 方法中调用

    class People {
    constructor(name) {
        this.name = name
        this.age = 20
    }
    sayHi() {
        console.log(this)
    }
}
const merry = new People("Merry")
merry.sayHi()

    People {name: “Merry”, age: 20}
