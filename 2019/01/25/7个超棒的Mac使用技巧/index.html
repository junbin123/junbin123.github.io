7个超棒的Mac使用技巧

2019-01-25
Mac

使用苹果电脑不仅仅因为优雅的设计，更因为其诸多便利，给生产者带来超高的工作效率。

下面分享 Mac 电脑的几个实用技巧。

1️⃣简单快速的分屏功能

日常使用电脑，如果需要频繁在两个软件之间进行劳作，那么分屏无疑适合你。

比如需要一边查找资料，一边打字。

Mac 电脑自带的分屏功能，就足以满足很多普通用户的需求。

方法也简单，只需鼠标长按任何软件左上角的小绿点👇

屏幕就一分为二了！

善用电脑的分屏功能，不仅可以大大提高工作效率，而且最大程度利用屏幕空间。

2️⃣好用的「三指拖移」手势

要细数 Mac 和 Windows 的区别，那么触控板丰富的手势操作就不得不提。

其中三指拖移就是非常好用的功能，

只需三个手指放在触控板上，就可以拖拽任何东西，相当于点按的效果。

比如要保存一张网页的图片👇

找回这个功能也简单，可以在「系统偏好设置」-「辅助功能」-「鼠标与触控版」-「触控版选项」中开启。

3️⃣快速输入 emoji 表情

作为一门通用语言，emoji 表情😎很多时候比文字要来的更直接有力。

如何在聊天中快速输入 emoji 表情，对于 Mac 用户完全不是问题。

只需使用 control + command + 空格键 ，就可以快速调用出 emoji 表情。

另外，Mac 还有其他特殊符号的快捷键输入👇

符号 快捷键
™ option + 2
∞ option + 5
• option + 8
≠ option + =
emoji 表情 control + command + 空格键
大写字母 shift + 字母
√ option + v

4️⃣显示文件扩展名

一部分用户可能有这个需求，需要随时查看文件的扩展名。

一般情况我们是通过显示简介来查看，但显然太慢了。

这时候，只需打开 Finder（访达），在「偏好设置」-「高级」-「显示所有文件扩展名」中开启。

5️⃣隐藏其他所有窗口

当你打开太多软件，导致桌面变得很乱，那么有没有什么办法可以快速隐藏掉其他窗口？

记住，是隐藏，不是关闭。

当然有啦，以 Chrome 为例，每个软件都有这个选项「隐藏其他应用」。

隐藏后就是这样的👇

6️⃣关于软件的关闭/退出/强制退出

新手有时会对关闭软件存在疑惑，比如点击左上角的红色圆点🔴，只是关闭了软件窗口而已，并非真正退出软件。

下面讲讲 Mac 电脑的关闭、退出、强制退出的效果和快捷键。

关闭（⌘ + W）：关闭当前的软件窗口（软件并没有真正退出进程），相当于点了左上角的🔴。

退出（⌘ + Q）：真正退出软件。

强制退出（⌘ + option + esc）：强制退出某个软件，通常在软件无响应时使用。

7️⃣批量命名文件

Mac 电脑对于批量命名文件也非常简单，只需选中所有要命名 的文件，右键选中「给11个项目重新命名」。
