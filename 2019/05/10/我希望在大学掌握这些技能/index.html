    我希望在大学掌握这些技能
    2019-05-10 · 吾儿滨滨

    大学生活是一人一生中为数不多可以肆无忌惮的时光，至少工作后，这种感觉会更加明显。

    我作为一个为数不少的废材大学生，读书期间，为所欲为地挥霍着自己的青春，大学生活过得不要太爽，甚至到了目无王法的地步。

    或许你们不知道，在中国，不努力学习，可能涉嫌违法👇。

    所以，真是这样的话，那么，我大概不应该在「外面」，而应该在「里面」。

    幸好人民政府宽容，没追究我的违法行为。但时间是公平的，大学不努力，工作两行泪。

    今天这篇文章会分享我为数不多的工作经历中，感受最深的三个工作技能，如果这些技能在大学早早 get 到，你同样可以在工作中为所欲为，无法无天。

    一、盲打 + 双拼打字

    打字速度太重要了！所以必须排在第一个。

    只要你活在这个现代社会，那么几乎就离不开打字。

    小到平时聊天回复消息，发发朋友圈动态；大到工作写写文档，做奇怪的会议记录。这都需要通过打字来完成。

    尤其在这个讲究效率的时代，只有拥有了无与伦比的打字速度，才会知道打字快究竟会带来多大的好处。

    那么问题来了，如何提高打字速度呢？

    两条途径，一是提高敲键盘的速度，二是减少敲键盘的次数。

    但聪明人不会二选一，聪明人全都要。

    幸运的是，社会已经给出了成熟的解决方案：盲打 + 双拼打字。

    先说说盲打。

    很多人都通过了网聊，无师自通学会了这项技能，甚至有时候，他们自己都没意识到，原来我打字这么快！

    直到有一天，他们看到一个人，笨拙地敲打着键盘时，那一刻，他们心中满是欢喜。

    但习得这项技能并不难，就是打字的时候，有意识地通过手指的肌肉记忆去寻找按键的位置，抑制自己看键盘的冲动。重复时间一长，自然而然就会发现，你再也不需要频繁去看键盘，可以更加专注于内容。

    再说说双拼打字，如果说学会了盲打可以让你骄傲，那么学会双拼可以让你目中无人。

    我先简单说一下双拼是个什么东西吧。

    首先，双拼不是一款应用，而是一个拼音方案，主流的输入法都可以切换到双拼输入法。很多人以为双拼是一个 App，兴冲冲去应用商店下载😂。

    双拼是相对全拼而言，使用双拼，每个字你只需敲击键盘两次，就可以打出来。

    直接从根源上提高了整体打字的效率，但学习双拼的路径比较陡峭，至少我是这样的。

    开始会各种不适应，心想，这是什么鬼！感觉几十年的打字经历受到了侮辱。

    但请相信我，只需坚持一个星期，你就会跟我一样，爱上这种打字方式。

    OK！限于篇幅，对于这个技能感兴趣，欢迎留言告诉我鸭，我考虑要不要写个具体的攻略。

    二、Excel 技能

    作为 Office 软件三大金刚之一，Excel 的重要性远高于 PPT 和 Word。

    我看来就是这样的，工作中使用 PPT 和 Word 更多是为了输出一份对外展示的报告，考验更多是一个人的逻辑是否清晰，审美是否合格。而不是工具使用是否熟练（当然，熟练使用固然好，但时间它不允许鸭）。

    而 Excel 不一样，Excel 拥有强大的数据处理能力，你思路清晰还不够，必须对工具非常熟练。

    但现实是，我们在学校用 Word 和 PPT 的次数远远高于 Excel。

    学校教的 Excel 知识比较浅，应付 Office 计算机二级还可以，可一到工作，就 GG 了。

    原因是实际处理数据工作中，需要的是「组合拳」，不是单一的知识点。

    所以，系统学习 Excel 就太重要啦，不然你永远无法体会到它的魅力。

    这里分享一个可行的学习路径：基础技巧 → 公式 → 数据透视 → 图表 → 进阶的 VBA。

    三、一点点编程基础

    我们这一代人，如果不会点编程，将来会不会就像父辈那一代人一样，不会英语呢（虽然我英语还是很菜）？

    别紧张，我要介绍的第三个技能——编程，不是为了要让你成为程序员。

    而是希望，当你遇到大量重复工作的时候，一定要有意识去思考更简洁、更快速的做法。

    而编程就是实现你这种懒惰的想法最好的工具。

    对于完全没有编程基础的小白，直接上手 Python 就完事了。

    之前为了研究用户对产品的评价，花了一个多星期，学习了一点 Python 爬虫的技能，爬取了微博上千条评论。

    如果当初手动去一条条复制，你知道的，那是一件多么难过的事情。

    如果你 5 分钟能完成 8 小时的工作量，这就相当于你坐飞机，别人还在翻山越岭慢慢地走。你说，气不气人！
