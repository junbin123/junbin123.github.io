《聪明工作法》笔记整理

这是少数派出品的课程，课程制作精美，短小精悍，实打实的工作场景，这里我做了一份笔记，学习过课程的同学往下看。

任务管理

  • 双列表系统
    • 今天必须做完的事情
    • 对自己很重要的事情：规划人生的三阶段
  • 三种任务类型
    • 个人复杂任务（LTF 体系）
      • List 列表
      • Tag 标签（含时间属性）
      • Filter 过滤
    • 团队任务（看板）
    • 数据型任务（Excel）

邮件礼仪和管理

职场邮件礼仪

  • 邮箱名称：`wujubin@163.com`
  • CC 和 BCC 的区别
    • CC（抄送）：对方知道就行，无需回应
    • BCC（密送）：收件人互相隐藏，避免群发的尴尬
  • 邮件内容和顺序
    1. 邮件正文（ABC）:行动 - 背景 - 结束语
    2. 邮件标题（类型 + 内容）：【通知】关于本月工资发放说明
    3. 写邮件顺序：附件 - 正文 - 标题 - 收件人
    4. 附件命名：修改日期、版本号、序列号
  • 自动回复：原因 + 联系

    个人邮件管理

  • 筛选邮件（FAST 法则）
    1. Filter 过滤
    2. Archieve 归档
    3. Snnoze 延后
    4. Transfer 流转
  • 邮件管理
    • 文件夹：生活、工作
    • 标签：来源、类型、项目

文件管理

  • 文件命名（时间 + 类型 + 内容 + 版本）：2018-09-12-通知-XX项目规划-第一版
  • 分类标准：找到属于自己的标准
    1. 按文件内容：学习、工作、生活、娱乐
    2. 按文件类型：音乐、文档、图片、视频
    3. 按工作内容：稿件、素材、资料、随笔
  • 查找工具：Everything、Droplt、Alfred
  • 云文件、协作办公
  • 重点管理：
    • 下载文件：及时处理
    • 桌面文件：不同分类的文件位置上做区别

数据收集

  • 流程：数据收集➡️数据处理➡️数据应用
    1. 数据收集（表单工具）：腾讯问卷、airtable（优秀）
    2. 数据处理：Excel
    3. 数据应用
  • 重点
    • 结构化收集信息
    • 按特定的规则整理数据

笔记整理

  • 通用原则
    1. 记重点：警惕完美主义
    2. 先收集：把笔记当成收件箱
    3. 可回溯：能容易找到
  • 使用 App，少使用纸币
  • 以文件夹为分类，辅助补充标签

思维导图

  • 应用广泛，可头脑风暴，可商务提案
  • 敢记最重要！
    1. 尽量记下听到的
    2. 不要只记自己的
    3. 不要太斟酌
    4. 要避免遗漏
  • 保证速度和内容
    • 纸张足够大
    • 从中间开始
    • 记录之间预留足够的空间
    • 只记录关键词
    • 自创常用标签

高效获取信息

  • 对信息源的控制
    • 获取信息的控制
    • 信息的质量：信息的有效时间和稀缺程度
    • 掌握信息
      • 软件推送
      • 主动搜索（Google）
    • 警惕回音效应：多角度获取
  • 处理信息的能力：速读 ➡️ 溯源 ➡️ 归纳

知识记忆内化

目的：从海量信息中获取有价值的部分，并内化为自己的知识

三个理论

必要难度

记忆中保持一定难度，长期对学习效果有效

  • 复述意识：康奈尔笔记
  • 间隔学习：不要一口气对一个知识点学太多
  • 费曼学习法
    1. 选择一个概念
    2. 讲给非专业人士听
    3. 发现理解不到位，返回原始材料
    4. 回顾和简化
  • 输出让你更强大

    遗忘曲线

    Anki 一个强大的工具

    记忆宫殿

  • 提升你的既视感
  • 调取更多感官参与
  • 联想解除记忆封印
    1. 根据数字的音义型进行预编码
    2. 将图片两两组合为故事
    3. 一边回顾故事，一边匹配数字
  • 在记忆宫殿里讲故事
