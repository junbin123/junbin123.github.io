【持续更新】数据分析实战笔记

建议搭配极客时间专栏《数据分析实战 45 讲》，一起食用，效果更佳，这是链接。

8️⃣自动化采集数据

一个数据的走势，是由多个维度影响的，通过收集更多维度的数据，同时保证数据质量，才能得到高质量的数据挖掘结果。

数据源分类

  1. 开放数据源：一般是针对行业的数据库，企业、高校、政府主导。
  2. 爬虫抓取：网站、App
  3. 日志采集：统计用户的操作；前端埋点，后端脚本收集、统计。
  4. 传感器：基本上采集物理信息；如图像、温度、湿度等。

如何使用开放数据源

  1. 单位维度：政府、企业、高校。
单位 数据源 网址
ImageNet 世界上最大的图像识别库，1500 万张图片 http://www.image-net.org/
北京大学 北大开放研究数据中心 http://opendata.pku.edu.cn/
  1. 行业维度：交通、能源、金融领域等。

使用爬虫抓取

基本过程：使用 Requests 爬取数据 → 使用 XPath 解析内容 → 使用 Pandas 保存数据

Python 爬虫利器：Selenium、Scrapy、PhantomJS、Puppteteer

不需要编程的爬取工具

火车头采集器：老牌采集工具，网页能看到内容都能抓取。

八爪鱼：含免费采集模板和云采集，云采集速度快，可自动切换 IP 地址，非常方便。

集搜客：可视化操作，所见即所得，没有云采集功能。

日志采集工具

日志即日记，它记录了用户访问网站的全过程，通过分析用户行为，提升系统性能，从而提高系统承载力。

日志采集的两种形式

  1. 通过 Web 服务器采集，例如 httpd、Nginx、Tomcat 都自带日志记录功能。
  2. 自定义采集用户行为，例如 JavaScript 代码监听用户行为、AJAX 异步请求后台日志记录。

埋点

埋点就是在有需要的的位置采集数据，进行上报。相当于一台摄影机。

市面有成熟工具：Google Analysis、Talkingdata、诸葛。

📊14. 数据可视化

商业智能分析

Tableau：国外软件、适合 BI 工程师、数据分析师。

PowerBI：微软出品，可与 Excel 无缝衔接。

可视化大屏

DataV：阿里巴巴出品。

FindReport：纯 Java 编写，商业报表软件。

前端可视化组件

概念

Canvas：2D、位图、相当于白板、HTML5 自带。

SVG：2D、矢量图、使用 XML。

WebGL：3D 绘图协议。

产品

ECharts：百度开源项目；基于 Canvas；与 DataV、Python 组合使用。

Three.js：基于 WebGL 框架。

AntV：蚂蚁金服出品；G2、G6、F2、L7。

D3：简化 JSC 操作难度。

Python 可视化

使用最多：Matplotlib、Seaborn（基于 Matplotlib 的高级可视化效果库）

其他：Bokeh、Plotly、Pyecharts、Mapbox

学习建议

  1. 重点推荐 Tableau，可先使用 DataV 上手。
  2. 掌握 Python 的必要。

《聪明工作法》笔记整理

这是少数派出品的课程，课程制作精美，短小精悍，实打实的工作场景，这里我做了一份笔记，学习过课程的同学往下看。

任务管理

  • 双列表系统
    • 今天必须做完的事情
    • 对自己很重要的事情：规划人生的三阶段
  • 三种任务类型
    • 个人复杂任务（LTF 体系）
      • List 列表
      • Tag 标签（含时间属性）
      • Filter 过滤
    • 团队任务（看板）
    • 数据型任务（Excel）

邮件礼仪和管理

职场邮件礼仪

  • 邮箱名称：`wujubin@163.com`
  • CC 和 BCC 的区别
    • CC（抄送）：对方知道就行，无需回应
    • BCC（密送）：收件人互相隐藏，避免群发的尴尬
  • 邮件内容和顺序
    1. 邮件正文（ABC）:行动 - 背景 - 结束语
    2. 邮件标题（类型 + 内容）：【通知】关于本月工资发放说明
    3. 写邮件顺序：附件 - 正文 - 标题 - 收件人
    4. 附件命名：修改日期、版本号、序列号
  • 自动回复：原因 + 联系

    个人邮件管理

  • 筛选邮件（FAST 法则）
    1. Filter 过滤
    2. Archieve 归档
    3. Snnoze 延后
    4. Transfer 流转
  • 邮件管理
    • 文件夹：生活、工作
    • 标签：来源、类型、项目

文件管理

  • 文件命名（时间 + 类型 + 内容 + 版本）：2018-09-12-通知-XX项目规划-第一版
  • 分类标准：找到属于自己的标准
    1. 按文件内容：学习、工作、生活、娱乐
    2. 按文件类型：音乐、文档、图片、视频
    3. 按工作内容：稿件、素材、资料、随笔
  • 查找工具：Everything、Droplt、Alfred
  • 云文件、协作办公
  • 重点管理：
    • 下载文件：及时处理
    • 桌面文件：不同分类的文件位置上做区别

数据收集

  • 流程：数据收集➡️数据处理➡️数据应用
    1. 数据收集（表单工具）：腾讯问卷、airtable（优秀）
    2. 数据处理：Excel
    3. 数据应用
  • 重点
    • 结构化收集信息
    • 按特定的规则整理数据

笔记整理

  • 通用原则
    1. 记重点：警惕完美主义
    2. 先收集：把笔记当成收件箱
    3. 可回溯：能容易找到
  • 使用 App，少使用纸币
  • 以文件夹为分类，辅助补充标签

思维导图

  • 应用广泛，可头脑风暴，可商务提案
  • 敢记最重要！
    1. 尽量记下听到的
    2. 不要只记自己的
    3. 不要太斟酌
    4. 要避免遗漏
  • 保证速度和内容
    • 纸张足够大
    • 从中间开始
    • 记录之间预留足够的空间
    • 只记录关键词
    • 自创常用标签

高效获取信息

  • 对信息源的控制
    • 获取信息的控制
    • 信息的质量：信息的有效时间和稀缺程度
    • 掌握信息
      • 软件推送
      • 主动搜索（Google）
    • 警惕回音效应：多角度获取
  • 处理信息的能力：速读 ➡️ 溯源 ➡️ 归纳

知识记忆内化
