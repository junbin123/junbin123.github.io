    CSS 实现水平垂直居中之方法总结
    2020-01-13 · 吾儿滨滨

    水平垂直居中是 CSS 布局常常遇到的问题，同时也是面试常考的。

    查阅了很多文章，实现的方法有很多，但比较杂。

    因此，本文将分别从行内元素和块状元素出发，总结各自的实现方案。

    1. 行内元素

    对于行内元素（inline、inline-block），水平居中很好实现，只需对其父元素设置text-align: center。

    而实现垂直居中，有两种方案：

    1.1 使用 line-height

    将父元素的line-height属性设置为高度，此方案需要知道父元素的高度。

    <style>
.parent {
  width: 400px;
  height: 100px;
  background: gray;
  text-align: center;
  line-height: 100px;
}
.children {
  display: inline;
  background: red;
}
</style>
<div class="parent">
    <div class="children">行内元素</div>
</div>
    1.2 使用 vertical-align

    为父元素设置display: table-cell和vertical-align: middle属性，此方案无需知道父元素的高度。

    <style>
  .parent {
    width: 400px;
    height: 100px;
    background: gray;
    display: table-cell;
    text-align: center;
    vertical-align: middle;
  }
  .children {
    display: inline;
    background: red;
  }
</style>
<div class="parent">
  <div class="children">行内元素</div>
</div>

    上面两种方案实现的效果是一样的。

    2. 块状元素

    对于块状元素，如果只想实现水平居中，一个常见的方法就是，设置margin: 0 auto。

    <style>
  .parent {
    width: 400px;
    height: 200px;
    background: gray;
  }
  .children {
    margin: 0 auto;
    width: 100px;
    height: 100px;
    background: red;
  }
</style>
<div class="parent">
  <div class="children"></div>
</div>

    如果需要水平垂直居中，实现方式大致可以分成两种。

    可以提前说明的是，下面介绍的具体方案，最终呈现的效果是一样的。

    2.1 绝对定位

    利用绝对定位，第一步将被居中元素的 top 和 left 属性均设为 50%，此时，元素的左上角位于父元素的中心点。

    第二步需要将元素拉回自身高度和宽度的一半。

    有三种方法可以实现第二步。

    ①如果知道被居中元素的高宽，可以直接使用 margin 属性。

    <style>
  .parent {
    width: 400px;
    height: 400px;
    background: gray;
    position: relative;
  }
  .children {
    width: 100px;
    height: 100px;
    background: red;
    position: absolute;
    top: 50%;
    left: 50%;
    margin: -50px 0 0 -50px;
  }
</style>
<div class="parent">
  <div class="children"></div>
</div>

    ②第二种方法也是利用 margin，但不需要知道被居中元素的高宽。

    <style>
  .parent {
    width: 400px;
    height: 400px;
    background: gray;
    position: relative;
  }
  .children {
    width: 100px;
    height: 100px;
    background: red;
    position: absolute;
    margin: auto;
    top: 0;
    bottom: 0;
    left: 0;
    right: 0;
  }
</style>
<div class="parent">
  <div class="children"></div>
</div>

    ③利用 CSS3 的 transfrom，也能在不知道元素高宽的情况下实现居中。

    <style>
  .parent {
    width: 400px;
    height: 400px;
    background: gray;
    position: relative;
  }
  .children {
    width: 100px;
    height: 100px;
    background: red;
    position: absolute;
    top: 50%;
    left: 50%;
    transform: translate(-50%, -50%);
  }
</style>
<div class="parent">
  <div class="children"></div>
</div>
    2.2 Flex 布局

    有别于传统的文档流布局，使用 Flex 布局，可以轻松实现元素的水平垂直居中。

    <style>
  .parent {
    width: 400px;
    height: 400px;
    background: gray;
    display: flex;
    align-items: center; 
    justify-content: center;
  }
  .children {
    width: 100px;
    height: 100px;
    background: red;
  }
</style>
<div class="parent">
  <div class="children"></div>
</div>

    总结

    最后，CSS3 transform 属性和 Flex 布局固然好用，但在实际项目中，需要考虑兼容问题。

    例如，IE8 不支持 transform 属性。

    而浏览器对于 Flex 布局的支持，要更严格。
